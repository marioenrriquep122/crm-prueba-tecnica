# CRM - Prueba Técnica

## 🚀 Tecnologías usadas
- Backend: Django + Django REST Framework
- Frontend: Next.js + TailwindCSS
- Base de datos: SQLite

---

## ⚙️ Backend

### Instalación
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

### Ejecutar servidor
python manage.py runserver

---

## 💻 Frontend

### Instalación
cd frontend
npm install

### Ejecutar servidor
npm run dev

---

## 🔐 Variables de entorno

Backend:
- SECRET_KEY
- DEBUG

Frontend:
- NEXT_PUBLIC_API_URL
